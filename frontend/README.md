# Team Task Manager Frontend

## Setup
1. Install dependencies:
   ```bash
   cd frontend
   npm install
   ```
2. Start the app:
   ```bash
   npm start
   ```

## Features
- Auth: Signup/Login
- Dashboard: task stats, overdue
- Projects: create/view (Admin), view (Member)
- Tasks: create/assign/update (Admin), update status (Member)
- Role-based access (Admin/Member)

## Deployment
- Ready for Railway (set REACT_APP_API_URL in Railway dashboard)
